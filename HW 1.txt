1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделанно предварительно?
- Сначала нужно скопировать ключ типа HTTPS 
- зайти в Git рописать команду git clone (ключ HTTPS)

2.Как создать git репозиторий из локального каталога на своем комьютере. Как его отправить на github.com?
- Зайти в Git 
- Создать каталог в котором будет репозиторий и прописать команду git init
- Создать какойто файл 
- Добавить его add .
- Закомитить commit -m "..."

3. Как получить на локальный компьютер изменения с github.com
- Скопировать силку на HTTPS 
- Зайти в Git
- Прописать команду git clone (HTTPS)

-git status отображает состояние рабочего каталога и раздела проиндексированных файлов
-git log перечисляет коммиты, сделанные в репозитории в обратном к хронологическому порядке СВЕРХУ В НИЗ
-git remote -v Выводит список всех подключенных удаленных репозиториев
